# Library system
a library system made in Python using classes and a little of system architecture for learning purposes

# What the project does
- The system let the user borrow a book, return a book, and see all the books available or loaned on system
- I used classes to keep a mold about users and books
- The project let the user find books by author and find users. However, the user search is only possible by the user id


# Technologies used
- Python 3.14

# How to use

- Clone this repo: https://github.com/sesgabriel/library-system
- Enter the project's folder (library-system)
- Run the program

# Usage example

You can use it from many ways like:
- adding a book
- creating a user
- deleting a user
- returning a book
- and more
